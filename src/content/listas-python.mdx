## ğŸ§ª 12. Ejemplo prÃ¡ctico: ordenando una lista paso a paso

Ahora veamos un ejemplo mÃ¡s completo donde usamos:

* listas
* `len()`
* bucles `for`
* comparaciÃ³n
* intercambio de valores

Este cÃ³digo ordena una lista de menor a mayor:

```python
lista = [1, 3, 2, 5]

for i in range(len(lista)):
    for j in range(i+1, len(lista)):
        if lista[i] > lista[j]:
            aux = lista[i]
            lista[i] = lista[j]
            lista[j] = aux

print(lista)
```

### ğŸ” Â¿QuÃ© hace este cÃ³digo?

1. **Definimos la lista:**

   ```python
   lista = [1, 3, 2, 5]
   ```

2. **Primer `for`: recorre los Ã­ndices de la lista**

   ```python
   for i in range(len(lista)):
   ```

   * `len(lista)` â†’ 4, porque la lista tiene 4 elementos.
   * `range(len(lista))` â†’ `range(4)` â†’ genera: `0, 1, 2, 3`.
   * Entonces `i` va tomando estos valores: 0, 1, 2, 3.

   `i` representa la **posiciÃ³n actual** desde donde vamos a comparar.

3. **Segundo `for`: compara con los elementos siguientes**

   ```python
   for j in range(i+1, len(lista)):
   ```

   * Empieza en `i + 1`, es decir, en el elemento que estÃ¡ **despuÃ©s** de `i`.
   * AsÃ­ evitamos comparar un elemento consigo mismo y repetidamente hacia atrÃ¡s.

   Por ejemplo:

   * Si `i = 0`, entonces `j` va de `1` a `3`.
   * Si `i = 1`, `j` va de `2` a `3`.
   * Si `i = 2`, `j` va de `3` a `3` â†’ solo `j = 3`.

4. **La comparaciÃ³n:**

   ```python
   if lista[i] > lista[j]:
   ```

   * Compara los valores en las posiciones `i` y `j`.
   * Si el valor en la posiciÃ³n `i` es **mayor** que el de `j`, estÃ¡n â€œal revÃ©sâ€ para un orden ascendente.
   * Entonces los intercambiamos.

5. **Intercambio usando una variable auxiliar**

   ```python
   aux = lista[i]
   lista[i] = lista[j]
   lista[j] = aux
   ```

   Esto hace el famoso â€œswapâ€:

   * Guardamos el valor de `lista[i]` en `aux`.
   * Ponemos el valor de `lista[j]` en `lista[i]`.
   * Luego ponemos el valor guardado en `aux` dentro de `lista[j]`.

   AsÃ­ no se pierde ningÃºn valor.

---

### ğŸ” Mini recorrido de ejemplo

Partimos con:

```python
lista = [1, 3, 2, 5]
```

* **i = 0** â†’ `lista[i]` es `1`

  * `j = 1` â†’ comparamos `1` y `3` â†’ no se cambia (1 < 3)
  * `j = 2` â†’ comparamos `1` y `2` â†’ no se cambia (1 < 2)
  * `j = 3` â†’ comparamos `1` y `5` â†’ no se cambia (1 < 5)

  Lista sigue igual: `[1, 3, 2, 5]`

* **i = 1** â†’ `lista[i]` es `3`

  * `j = 2` â†’ comparamos `3` y `2` â†’ **3 > 2**, entonces se intercambian
    Lista pasa a ser: `[1, 2, 3, 5]`
  * `j = 3` â†’ comparamos `3` y `5` â†’ no se cambia (3 < 5)

* **i = 2** â†’ `lista[i]` es `3`

  * `j = 3` â†’ comparamos `3` y `5` â†’ no se cambia (3 < 5)

* **i = 3** â†’ ya estÃ¡ en el Ãºltimo valor, no habrÃ¡ cambios.

Resultado final:

```python
print(lista)  # [1, 2, 3, 5]
```

Este es un ejemplo simple de algoritmo de ordenamiento, ideal para que un estudiante entienda:

* cÃ³mo recorrer listas con Ã­ndices
* cÃ³mo comparar elementos
* cÃ³mo intercambiar valores (`swap`)
